## Deployment Object
- concept:
  - deploy pod/s
  - upgrade (upgrade all pods together)
  - rollout (upgrade pods one after another)
  - rollback
  - pause and resume
- yaml is same as rs
  - just update kind to deployment object.

- command  
  - kubectl get **deployments**
  - kubectl get **all**
